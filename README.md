# Edflo | Campus Operating System

Welcome to the landing page repository for **Edflo**, the modern operating system designed for educational campuses.

## Overview

Edflo is a platform created to simplify and consolidate the digital infrastructure of modern universities and schools. By unifying attendance tracking, academic records, and campus administration into a single, cohesive interface, Edflo eliminates the friction of managing fragmented legacy systems.

This landing page serves as the primary marketing and waitlist portal for the upcoming beta release.

## Design Aesthetic

The landing page is built with a premium, tech-focused "System Visualizer" aesthetic. It leans into a highly modern design language featuring:

- **Glassmorphism & Depth**: Soft canvas backgrounds paired with subtle blurs and architectural glows.
- **Tech-Widget Layouts**: Information presented in dashboard-style bento grids with mono-font "system status" tags.
- **Pixel-Art Elements**: Geometric, pixel-art inspired shapes (like the signature Edflo logo dots) used throughout as decorative and functional UI accents.
- **Fluid Motion**: Smooth, staggered entrance animations powered by Framer Motion.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
